Propósito del sitio

Elaborar recursos educativos digitales para dinamizar las estrategias de aprendizaje. Una estrategia la constituyen los Objetos virtuales de aprendizaje y se va a diseñar cinco entornos Virtuales de aprendizaje que expliquen el funcionamiento y el proceso para el diseño de sitios web. En este caso se resalta el tema 1: Etiquetas Básicas de HTML, atributos y valores de un sitio (página) web.

El hipertexto (HTML) es un lenguaje informático que forma la mayoría de las páginas web y plataformas online. Los usuarios web pueden crear y estructurar secciones, párrafos y enlaces usando elementos, etiquetas y atributos.Una etiqueta contiene mucha información. Le indica al navegador dónde empieza y dónde termina cada elemento (valor), mientras que un atributo describe las características.
