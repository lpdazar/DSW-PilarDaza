El propósito de esta página Web

Desde que comenzó la pandemia del COVID-19, la educación en todos los institutos educativos ha reemplazado la presencialidad a la educación virtual, utilizando variedad de programas para enseñar online y aplicaciones de video llamadas. Esto ha traído como resultado la necesidad de elaborar recursos educativos digitales para dinamizar las estrategias de aprendizaje. Por lo tanto, para la facilidad de los estudiantes de la Universidad del estado, se creó esta página web para dar un curso básico que constituye la creación de paginas web que hace parte del programa de Ingeniería de sistemas. La idea es que sea accesible para todos y fácil de entender.

El diseño de páginas web es la construcción de documentos de hipertexto para su visualización en diferentes navegadores. Así como asignarle una presentación para diferentes dispositivos de salida (en una pantalla de computador, en papel, en un teléfono móvil, etc). Estos documentos o páginas web pueden ser creadas: Con archivos de texto en HTML, Asp, Aspx, JavaScript, JSP, Python, Ruby, tambien utilizando un programa visual WYSIWYG o WYSIWYM de creación de páginas o usar un Script del lado del servidor, para generar la página web.

Introducción

La empresa HtmlDesign ha sido contratada por la universidad del Estado para diseñar cinco entornos virtuales de aprendizaje, los cuales tiene los siguientes contenidos en la área Diseño de Sitios Web:

  • Tema 1: Etiquetas Básicas de HTML, atributos y valores
  • Tema 2: Sintaxis y Selectores CSS, Interacción Humano Ordenador
  • Tema 3: Etiquetas semánticas y CSS Grid
  • Tema 4: Flex Box, Listas y enlaces
  • Tema 5: Hojas de Estilo en Cascada CSS y Frameworks CSS
